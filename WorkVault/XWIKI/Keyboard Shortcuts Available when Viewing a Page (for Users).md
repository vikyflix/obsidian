## Keyboard Shortcuts Available when Viewing a Page (for Users)

## Reference

You can use keyboard shortcuts to use some XWiki features. Keyboard shortcuts help you save time since you never have to take your hands off the keyboard to use the mouse. You need to [view a Page](https://www.xwiki.org/xwiki/bin/view/Doc/xs/user/base/page/view-page/) to be able to use the following keyboard shortcuts:

## FAQ

## Why can't I use the keyboard shortcuts?

View keyboard shortcuts are only available to [advanced users](https://www.xwiki.org/xwiki/bin/view/Doc/xs/user/base/page/edit-page/simple-advanced/#HAdvancedMode), preventing [simple users](https://www.xwiki.org/xwiki/bin/view/Doc/xs/user/base/page/edit-page/simple-advanced/#HSimpleMode) to get unexpected behavior when pressing keyboard keys by mistake.

## Are there keyboard shortcuts that are available only for administrators?

Yes, there are some [keyboard shortcuts available only for administrators when viewing a Page](https://www.xwiki.org/xwiki/bin/view/Doc/xs/admin/base/page/keyboard-shortcuts-viewing-page/).
