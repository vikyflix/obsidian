## Reference

As an administrator, you have access to additional [keyboard shortcuts that are available when viewing a Page](https://www.xwiki.org/xwiki/bin/view/Doc/xs/admin/base/page/keyboard-shortcuts-viewing-page/#). These are the following:

| Keyboard Press | Description |
| --- | --- |
| F2 | Opens the wizard for [renaming the Page](https://www.xwiki.org/xwiki/bin/view/Documentation/UserGuide/Features/DocumentLifecycle/#HMove2FRename). |
| Delete | Opens the wizard for [deleting the Page](https://www.xwiki.org/xwiki/bin/view/Documentation/UserGuide/Features/DocumentLifecycle/#HDelete) |
